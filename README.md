# 結 · 명식 나침반 🧭

사주명리 기반 리딩 웹페이지. 개인 리딩과 자동 생성기로 구성됨.

| 파일 | 설명 | 배포 후 주소 |
|---|---|---|
| `index.html` | **이유진 개인 리딩** (완성본) | `https://{내아이디}.github.io/{repo이름}/` |
| `hojin.html` | **최호진 개인 리딩** (삼주 기준, 결혼운·재물·이동수·궁합 포함) | `.../{repo이름}/hojin.html` |
| `agent.html` | **사주 리딩 에이전트** ⭐ — 개인 리딩 + 궁합 + 자동 QA검증 + 최근 조회 기억 | `.../{repo이름}/agent.html` |
| `template.html` | 간단 자동 생성기 (개인 리딩만) | `.../{repo이름}/template.html` |

### 🤖 agent.html 이 메인이에요
방금까지 진행한 전체 흐름(입력 → 사주 자동계산 → 리딩 → 궁합 → **계산 오류 검증**)을 하나로 묶은 버전:
- **개인 리딩 / 궁합 보기** 두 모드
- **🔍 자동 QA 검증 리포트** — 계산이 맞는지 스스로 점검 (일주 앵커 self-test, 오행 합계, 십성 일관성, **절기 경계 경고**, 대운수 범위, 시간 미상 여부) → A/B/C 신뢰등급 표시
- **기억 기능** — 최근 조회한 사주 6명을 저장(브라우저), 클릭해서 다시 불러오기
- 궁합은 일간 생극(누가 누구를 품는지)·배우자궁 합충·기운 보완까지 자동 분석

> ⚠️ QA가 왜 중요하냐면 — 만드는 과정에서 실제로 일주 계산이 어긋난 적이 있었어요(기준일 오류). QA의 '앵커 self-test'가 바로 그런 오류를 자동으로 잡아줍니다.

## 🚀 GitHub Pages로 무료 배포하는 법 (5분)

> 계정·비밀번호가 필요한 작업이라 직접 해야 해. 아래 클릭만 따라오면 됨.

1. **깃허브 가입/로그인** → https://github.com
2. 오른쪽 위 **`+` → New repository**
3. 설정:
   - Repository name: `saju` (아무 이름이나)
   - **Public** 선택 (Pages는 public이어야 무료)
   - **Create repository** 클릭
4. 만들어진 repo에서 **`Add file` → `Upload files`**
   - `index.html`, `hojin.html`, `agent.html`, `template.html`, `README.md` 를 **드래그해서 올리기**
   - 아래 **Commit changes** 클릭
5. 상단 **Settings → 좌측 Pages** 메뉴
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` / `/ (root)` 선택 → **Save**
6. 1~3분 기다린 뒤 새로고침하면 상단에 주소가 뜸:
   - 개인 리딩 → `https://{내아이디}.github.io/saju/`
   - 호진 리딩 → `https://{내아이디}.github.io/saju/hojin.html`
   - 자동 생성기 → `https://{내아이디}.github.io/saju/template.html`

이 주소를 인스타·카톡 등 어디든 공유하면 됨. 🎉

## 📎 팁

- **주소 짧게/예쁘게** 하고 싶으면 repo 이름을 `{내아이디}.github.io`로 만들면 주소가 `https://{내아이디}.github.io/` 로 깔끔해짐.
- **수정**은 GitHub에서 파일 열고 연필(✏️) 아이콘 → 고치고 Commit 하면 자동 반영.
- **커스텀 도메인**(예: `mysaju.me`)도 Settings → Pages에서 연결 가능.

## ⚙️ template.html 작동 방식

- 순수 HTML/JS 단일 파일 (서버·DB 없음, 브라우저에서 다 계산)
- 입력: 이름·성별·양력 생년월일·시각
- 계산: 사주 원국(년월일시 간지) → 오행·십성 → 신강/신약 → 지지 충 → 대운(순/역행·대운수) → 세운
- 출력: 성격 / 돈 버는 법 / 재물·일 / 이동수 / 관계 / 대운표 / 세운표 / 요약

### 정확도 안내
- **일주**는 정밀 계산(1998-02-22=庚子 기준 앵커로 검증).
- **절기(월주)**는 근사 고정값이라, 각 절기 경계(±1일)에 태어난 사람은 월주가 어긋날 수 있음.
- 어디까지나 **엔터테인먼트·자기성찰용**. 미래를 확정·보장하지 않음.
